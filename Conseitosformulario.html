Conceitos

Formulários HTML são um dos principais pontos de interação entre um usuário e um site ou aplicativo. Eles permitem que os usuários enviem dados para o site. Na maior parte do tempo, os dados são enviados também para o servidor da web, mas uma página da web pode ser interceptada para usá-los por conta própria. Um formulário HTML é feito de um ou mais widgets. Esses widgets podem ser campos de texto (linha única), caixas de seleção ou de várias linhas, caixas de seleção ou botões de rádio. A maior parte do tempo, estes elementos são essenciais com uma legenda que descreve o seu objetivo.
